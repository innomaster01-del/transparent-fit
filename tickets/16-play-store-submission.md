# T6.2 · Play Store Submission

**Phase:** 6 · **Estimate:** 2 days

## Goal

Public release on Google Play.

## Steps

1. **Assets:**
   - Hi-res icon (512×512 PNG)
   - Feature graphic (1024×500)
   - Screenshots: at least 2 per form factor (phone, 7" tablet, 10" tablet)
2. **Listing:**
   - Short description (80 chars)
   - Full description (4000 chars)
   - App category
3. **Content rating:** complete IARC questionnaire → Everyone
4. **Data safety:**
   - "No data collected, no data shared"
   - List permissions and their purposes
5. **Sign with the same upload key** used for internal testing
6. **Production track** → submit → review (~24 hours)

## Acceptance

- [ ] App live on Play Store
