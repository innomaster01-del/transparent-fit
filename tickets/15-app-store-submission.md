# T6.1 · App Store Submission

**Phase:** 6 · **Estimate:** 3 days

## Goal

Public release on the iOS App Store.

## Steps

1. **Assets:**
   - Icon set (1024×1024 master, all sizes auto-generated by Xcode)
   - Screenshots: 6.7" iPhone (1290×2796), 6.5" iPhone, 5.5" iPhone, 12.9" iPad
   - Optional: app preview video (15–30s)
2. **Metadata:**
   - App name + subtitle (max 30 chars each)
   - Description (4000 chars)
   - Keywords (100 chars total)
   - Support URL, marketing URL, privacy policy URL
3. **Privacy:**
   - Camera + Photo Library data declarations
   - "Data not collected" — since we don't send anything off device
4. **Pricing:** Free (decide later if/how monetization happens)
5. **Categories:** Lifestyle (primary), Shopping (secondary)
6. **Age rating:** 4+ (no objectionable content)
7. **Submit for review** — Apple takes 1–3 days typically

## Likely review questions

- "Where is the user's data sent?" → Nowhere, all on-device
- "Why do you need photo library write?" → Save snapshots
- "Why microphone permission?" → Required by Vision Camera even when audio disabled

## Acceptance

- [ ] App passes review
- [ ] Live on App Store
